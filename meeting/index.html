自动部署到GitHub

如何获取GitHub Personal Token：

  1. 登录GitHub，点击右上角头像 → Settings
  2. 左侧菜单选择 Developer settings → Personal access tokens → Tokens (classic)
  3. 点击 Generate new token → Generate new token (classic)
  4. 设置备注名称，选择过期时间，勾选 repo 权限
  5. 点击 Generate token，复制生成的token

使用说明

  1. 在编辑区域修改祷告会内容
  2. 配置GitHub部署信息（需要Personal Token）
  3. 点击"一键自动部署"将页面直接发布到GitHub Pages
  4. 访问生成的网址分享给教会同工
